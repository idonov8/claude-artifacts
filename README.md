# Claude Artifacts Gallery

A collection of interactive demos, tools, and experiments generated by [Claude](https://claude.ai), hosted as a browsable gallery via GitHub Pages.

## Live Site

**[https://idonov.github.io/claude-artifacts/](https://idonov.com/claude-artifacts/)**

## How It Works

- `index.html` — Gallery page that fetches `artifacts/manifest.json` and renders artifact cards
- `artifacts/manifest.json` — Registry of all artifacts (title, description, date, path)
- `artifacts/<slug>/index.html` — Each artifact is a fully self-contained HTML file

GitHub Pages serves the repo root directly — no build step required.

## GitHub Pages Setup

1. Go to **Settings → Pages**
2. Source: **Deploy from a branch** → `main` → `/ (root)`
3. Save — the site will be live at `https://<username>.github.io/claude-artifacts/`

> The repo must be public (or you need GitHub Pro for private Pages).

## Adding Artifacts

Claude adds artifacts automatically using the `artifact-gallery` skill. Each artifact is:
- A fully self-contained HTML file (no external dependencies)
- Registered in `manifest.json` with title, description, and date
- Committed alongside the manifest update
